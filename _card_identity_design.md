# Идентичность классов: колода ОТ пассива (С55)

Рефактор идентичности (шаг 5 главы 1). Контекст: сигнатурные классовые карты раньше
подбирались тематически (заложить StS-инфру). Входя в фазу контента — проектируем
**колоду ОТ пассива**. Память: [[starter-deck-reveals-passive]], [[replayability-doctrine]],
[[design-north-star]], [[berserker-redesign]], [[chapter-arc-narrative]].

---

## 0. ПРИНЦИП (юзер С55) — фильтр приёмки

Стартовая дека = классовые карты, которые **РАСКРЫВАЮТ пассив** (учат отдельным граням
движка), но **НЕ закрывают билд внутри одной деки**. Каждая сигнатурка — «учитель грани»,
открытый, а не звено замкнутого комбо. Билд ВОЗНИКАЕТ по ходу забега (драфт/ковка/награды),
пассив — сквозная нить, generic-пул — открытый холст.

**Тест приёмки каждой карты:** «рельсы или учитель?» Если карта+стартер дают замкнутый
самодостаточный комбо-луп — это рельсы, переделать.

**Анти-паттерн (пример юзера):** стартер Мага сейчас играет ПАР из коробки (Закипание +
Ignite + Splash = рабочее комбо вручено) = рельсы. → де-рельсить при переделе Мага/Воина.

## 1. БЕРСЕРК — с нуля (АКТИВНЫЙ ШАГ)

**Дефицит:** 0 классовых карт. Движок (`berserker.py` + `abilities/berserker.py`):
- `hp_overdraft=True` → уход в МИНУС HP (не умирает на 0);
- глубина минуса → множитель урона (`EffectCalculator` шаг 8-ter);
- победа «в коме» → `|долг|` → Forge Points (`on_combat_won`), выживает hp=1;
- способность «Безумие» → карты за 0 энергии ценой HP (`MADNESS_HP_PER_COST=4`).

**Направление A «Кровь-в-мощь» (юзер выбрал):** чистый хардкор, без спаскруга
(доктрина glass-cannon, [[balance-findings-berserker-glasscannon]] — стену НЕ лечим).

### Три сигнатурки = три грани движка (rails-чек пройден)

| Карта | Редкость | Грань | Эскиз (числа = ЗАГЛУШКИ, калибровка в капстоуне) |
|-------|----------|-------|--------------------------------------------------|
| **Кровавая ярость** | UNCOMMON | долг → урон | Атака. Урон = base + k×(глубина HP-долга). Чем глубже минус — тем больнее. Движок кат.4 в карте (роль Возмездия). От ЛЮБОГО источника долга → открыто. |
| **Безрассудный удар** | COMMON | корм Безумия | Дорогая мощная атака (cost 3, большой урон). Нормально дорогая → через Безумие каст за 0 ценой HP → нырок → множитель. Учит «Безумие = каст дорогого ценой HP». |
| **Жажда крови** | UNCOMMON | долг → FP | Атака ценой HP (в долг); при ДОБИВАНИИ врага часть долга сразу → FP (мини-пик). Учит петлю долг→ковка. Не комбо сам по себе. |

### Стартдек (de-railed)
- Сейчас: `strike×5 + heavy_blade×2 + flex + battle_cry` (всё generic).
- Станет: **+2 учителя в стартер** (Кровавая ярость + Жажда крови), убрать 2 strike →
  остаётся `strike×3 + heavy_blade×2 + flex + battle_cry + Кровавая ярость + Жажда крови`.
  3-я (Безрассудный удар) — в класс-пул драфта, НЕ в стартер. Замкнутого лупа нет:
  payoff’ы вознаграждают долг от любого источника, игрок сам ищет, как его генерить/держать.

### Технический план (1 коммит = подзадача; план→память→доки→код; baseline зелёный/re-bless)
1. **Эффект-кирпичи** (`core/cards/berserker.py`): `DebtScalingDamageEffect` (урон от
   глубины долга), `SelfHarmDamageEffect` (атака ценой HP в долг), `DebtToForgeOnKillEffect`
   (добивание → часть долга в FP). Чистые, через `EffectCalculator`. + юнит-тесты кирпичей.
2. **Три карты-фабрики** + реэкспорт `core/cards/__init__.py` + запись
   `CLASS_FACTORIES["Berserker"]` в `catalog.py`. Карты достижимы в драфте/наградах класса.
3. **Стартдек** `get_berserker_deck()` (+2 сигнатурки, −2 strike). Тесты состава деки.
4. **Сим-натив + re-bless baseline**: BerserkerPolicy видит новые карты (бот их играет);
   замер wall/ceiling (forge-on хвост p90), переблагословить эталон. Гард зелёный.
5. **Доки** (PATCHNOTES + _project_map + этот файл) + трекер + UI-смоук (карты рисуются,
   бейджи). Финал.

## 2. ВОИН — Дисциплина-как-ресурс (✅ ЗАВЕРШЕНО С56)

Реализовано (5 коммитов, шаблон Берсерка): кирпичи спендеров → фабрики+регистрация →
де-рельсенный стартер → сим-пилот+re-bless (wall 32→28, ceiling 51→51 СОХРАНЁН, forge
95→90%) → UI/доки. 979 тестов, гард 7/7. Числа = заглушки → капстоун-калибровка тройки.

Принято юзером С56 (вариант 3 «Дисциплина-как-ресурс» + развилка экономики
[[economy-axis-trinity]]). Воин = ось ВЫЖИВАЕМОСТИ, НЕ FP — никакой FP-генерации
(в отличие от Берсерка). Связь с золотом-выживаемостью придёт при раскатке эконом-дуги.

**Движок (вариант 3):** Дисциплина (`discipline`, стак в statuses) копится от обороны
(пассив: начал ход со щитом → +1; EffectCalculator шаг 2d: +N урона за стак ПОКА
держишь). Сигнатурки её **ТРАТЯТ** (сжигают стаки → разовый эффект, теряя +урон) —
выбор «копи ради стабильного +урона vs слей в пик». Нарастающая нестабильность вверх.

### Три грани (шаблон Берсерка, числа = ЗАГЛУШКИ → капстоун-калибровка)
| Карта | Ред. | Грань | Эскиз |
|-------|------|-------|-------|
| **Карающий строй** | UNC | Дисц → бурст | Трать ВСЮ Дисциплину → урон = base + k×(потрачено). Роль «Возмездия». |
| **Стена щитов** | UNC | Дисц → выживаемость | Трать ВСЮ Дисциплину → щит = base + k×(потрачено) (+мини-хил). Оборонный payoff = ось класса. |
| **Стойка** | COM | билдер | Щит + сразу +N Дисциплины. Учит «оборона = накопитель». |

NB: трата = обнулить стак (`set_status('discipline',0)`) ДО `calculate_damage`, чтобы
шаг 2d не задвоил (карта САМА — payoff накопленного). Урон/щит через EffectCalculator.

### Стартдек (de-railed)
- Сейчас: `strike×4 + defend×4 + heavy_blade + Возмездие + Стальной заслон` (11).
- Станет: `strike×4 + defend×4 + heavy_blade + Карающий строй + Стена щитов` (11).
  Возмездие/Стальной заслон/Бастион → класс-драфт-пул. **Стойка** (билдер) — тоже в
  ПУЛ, НЕ в стартер: так в стартере единственный источник Дисциплины = пассив
  (держи щит), а 2 спендера её тратят → максимально «открыто», не рельсы (нет
  в-стартере замкнутого генератор→payoff лупа; defend→пассив→спенд = ядро движка,
  учится честно).

### Техплан (1 коммит = подзадача; baseline зелёный/re-bless)
1. Эффект-кирпичи `core/cards/warrior.py`: `DisciplineBurstDamageEffect` /
   `DisciplineToShieldEffect` / `DisciplineGainEffect` + юнит-тесты.
2. Фабрики (Карающий строй/Стена щитов/Стойка) + `CLASS_FACTORIES["Warrior"]` += 3 +
   реэкспорт `__init__`.
3. Стартдек `get_warrior_deck` (+2 спендера, −Возмездие/−Заслон). Тесты состава.
4. Сим-натив `WarriorPolicy` (копи Дисциплину держа щит → трать спендером при пороге)
   + builds-эвристика + re-bless baseline. Гард зелёный.
5. Доки (PATCHNOTES + _project_map + этот файл) + трекер + UI-смоук.

## 2-bis. МАГ — «Гни»: Мастерство-перегруз + HP-гамбл (✅ ЗАВЕРШЕНО С56)

Реализовано (4 коммита): кирпичи (Разгон/Резонанс) → фабрики+регистрация → %-Нестабильность
(эскалир.)+%-хил → де-рельс стартдек+сим-пилот+re-bless. Эталон спущен: wall 30→18,
ceiling 56→35, forge 100→78% (де-рельс −6/−13 легитимно + HP-гамбл −6/−8 ожидаемо: HP-ось
недостроена → подрастёт сама). Движок ЖИВ (forge-reach без гамбла 98%). Юзер: эталон=полный
цикл, Разгон=чистый расход. 996 тестов, гард 7/7. **ДОЛГ:** ре-блеснуть ВВЕРХ при HP-оси.

Принято юзером С56 (обсуждение). Маг = ось **HP/казино** ([[economy-axis-trinity]]) —
ГЭМБЛИТ HP (не банкует валюту, в отличие от Берсерка→FP). НЕ генерит FP. Едет
самодостаточно (%-механики играбельны без инфры max HP; HP-ось = отдельная дуга после).

**Движок = Мастерство (держимый компаунд) + Нестабильность (HP-гамбл):**
- `mastery` копится +1/комбо (пассив `on_card_played_passive`), +N урона всем атакам
  (шаг 2c). На пороге ≥5 → ×1.5 «перегруз» (награда РАСТЁТ сама от большего Мастерства).
- **Контраст в тройке:** Воин ТРАТИТ Дисциплину · Маг ДЕРЖИТ-и-перегружает Мастерство ·
  Берсерк рискует жизнью.

**Движок-правки (Маг-локальные, %-масштаб-инвариантность):**
- **Нестабильность → эскалирующий %**: цена = `% от max HP/ход`, растёт с глубиной
  Мастерства за порогом (напр. base 4% + 1%/стак сверх 5). Было плоские −3 HP (протухало).
- **Комбо-хил → %** от max HP (было +7). Маг = HP-churn: %-искрит / %-лечится.
- Шаг 2c (×1.5) — БЕЗ изменений (эскалирует только ЦЕНА, награда растёт сама).

**3 грани (числа = ЗАГЛУШКИ → капстоун):**
| Карта | Грань | Эскиз |
|-------|-------|-------|
| **Тайное сосредоточение** (есть) | билдер | Мастерство +2 (чистый учитель) → ПУЛ |
| **Разгон** (нов., `OverclockEffect`) | гамбл | заплати X% HP → +N Мастерства разом (перешагни порог по воле) → СТАРТЕР |
| **Резонансный разряд** (нов., `MasteryScalingDamageEffect`) | глубина | урон = base + Мастерство×k (читает, НЕ тратит) → СТАРТЕР |

**Стартдек (de-railed):** было `strike×2 defend×3 Поджог Всплеск Закипание Тайное
сосредоточение`. Станет `strike×2 defend×3 Поджог Всплеск Разгон Резонансный разряд` (9).
**Закипание убрано** (ПАР пред-собран = рельсы); Поджог+Всплеск раздельно → ПАР собирает
игрок. Закипание/Стихийный всплеск/Тайное сосредоточение → драфт-пул.

**Техплан (шаблон Воина, 1 коммит = подзадача):**
1. Кирпичи `core/cards/mage.py`: `OverclockEffect` (заплати %HP→+Мастерство),
   `MasteryScalingDamageEffect` (урон+Мастерство×k) + юнит-тесты.
2. Фабрики (Разгон/Резонансный разряд) + `CLASS_FACTORIES["Mage"]` += 2 + реэкспорт.
3. Движок: %-Нестабильность (эскалир.) + %-комбо-хил в `mage.py` (+константы) + тесты.
4. Стартдек `get_mage_deck` (−Закипание/−Тайное сосред., +Разгон/+Резонансный разряд) +
   `MagePolicy` пилотирует (строй Мастерство → гамбл Разгоном → выжимай Разрядом, не
   суицидь) + builds + re-bless. Гард зелёный.
5. Доки + трекер + UI-смоук.
