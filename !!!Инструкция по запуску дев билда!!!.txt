Мануал по установке:

1. Установить Python 3.10 (Если у вас установлена версия 3.11\3.12 или более новая, она может не подойти и ее может потребоваться удалить. Версия 3.10 самая стабильная.)
-------------Ссылка на скачивание-------------
https://www.python.org/downloads/release/python-3100/
----------------------------------------------
Листаем в самый низ, скачиваем версию для windows. Разархивируем, устанавливаем.
!!!ВАЖНО!!!
При установке обязательно поставить галочку "Add Python 3.10 to PATH" в самом низу окна! Без этого консоль не поймет команды.

2. Открыть консоль cmd от имени администратора и ввести следующую команду:

pip install pygame pillow requests

Эти библиотеки необходимы для запуска.

3. Запускаем файл main.py двойным кликом ЛКМ
ИЛИ: ПКМ -> открыть с помощью -> Python
